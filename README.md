Clone it and Run it:

Create a .env.local file and give the following fiels
```
NEXT_PUBLIC_CLIENT_ID=
NEXT_PUBLIC_CLIENT_SECRET=
JWT_SECRET=
```
Then :-
```bash
npm i
npm run dev
```
